untitled
========

Embedded development targeting Game Boy Advance.

Building
--------

Building requires `devkitPro <https://devkitpro.org/>`_ installed and the
``DEVKITPRO`` and ``DEVKITARM`` environment variables set.

.. code:: shell

   ./configure.sh
   cmake --build build/
